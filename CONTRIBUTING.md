# Contributing to WinControl

Thank you for your interest in contributing!

## How to Contribute

1. **Fork the repository** and create your branch from `main`
2. **Make your changes** with clear, focused commits
3. **Test your changes** on Windows (native or WSL)
4. **Submit a pull request** with a clear description

## Development Setup

```bash
pip install pywin32 pillow mss
python server.py
```

## Code Style

- Follow PEP 8 for Python code
- Keep the server simple and focused
- Document new endpoints in SKILL.md

## Reporting Issues

- Use GitHub Issues for bug reports
- Include Windows version and Python version
- Provide steps to reproduce

## Questions?

Open an issue or reach out on the OpenClaw Discord.
